                                     AVIX Demo Applications
==================================================================================================
Description:
--------------------------------------------------------------------------------------------------
This directory contains an Extended Demo application.

>>> A global overview about functionality and usage of this demo can be found in 
    AVIX_Explorer16_ExtendedDemo.pdf present in this directory.

>>> A detailed description about the structure of this application can be found in 
    AVIX_ExchangeMechanism.pdf which can be found in the _AVIX\Doc directory.

NOTE: WHEN USING A FREE  DEMO DISTRIBUTION,  THE APPLICATION WILL RUN FOR A MAXIMUM  OF 30 MINUTES
      AFTER WHICH IT WILL STOP. THIS LIMITATION IS BUILT INTO THE AVIX FREE DEMO DISTRIBUTION!
--------------------------------------------------------------------------------------------------


Target board:
--------------------------------------------------------------------------------------------------
This demo is configured for the Microchip Explorer16 development board equiped with a 
PIC24FJ256GB210 MCU using the ICD3 debugger.

>>> When not having access to a ICD3 debugger, change to the type that is available.


=================================
How to run the demo with MPLAB8x:
=================================
1:	Open file \_dev_board_demos\_demo1_Explorer16_ExtendedDemo\extDemo.mcw
2:  Build the project
3:  Flash the program to the development board
4:  Run the application




===================================
How to run the tutorial with MPLABX
===================================
1: Import the MPLAB8x project file either from the MPLABX opening  screen or from the import  menu 
   entry in the MPLABX file menu.
   Project file: \_dev_board_demos\_demo1_Explorer16_ExtendedDemo\extDemo.mcP
2: Build the project
3: Flash the program to the development board
4: Run the application
